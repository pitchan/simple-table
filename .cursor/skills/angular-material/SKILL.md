# Angular Material Skill

## Overview
Build beautiful, accessible Angular applications using Angular Material component library with custom theming and responsive layouts.

## Topics Covered

- [Form controls (inputs, selects, checkboxes)](https://v16.material.angular.dev/components/form-field/overview)
- [Navigation (toolbar, sidenav, menus)](https://v16.material.angular.dev/components/sidenav/overview)
- [Navigation menu](https://v16.material.angular.dev/components/menu/overview)
- [Cards](https://v16.material.angular.dev/components/card/overview) 
- [Lists](https://v16.material.angular.dev/components/list/overview)
- [Grids](https://v16.material.angular.dev/components/grid-list/overview)
- [Data tables and pagination](https://v16.material.angular.dev/components/table/overview)
- [Dialogs and snackbars](https://v16.material.angular.dev/components/dialog/overview)
- [snackbars](https://v16.material.angular.dev/components/snack-bar/overview)

### Theming
- Custom theme creation
- Color palettes and typography
- Dark mode implementation
- Component customization
- CSS variables integration

### CDK (Component Dev Kit)
- [Overlay system](https://v16.material.angular.dev/cdk/overlay/overview)

- Performance optimization

- Angular Core fundamentals
- CSS/SCSS basics
- Component architecture

- Build consistent Material UI interfaces
- Implement accessible components
